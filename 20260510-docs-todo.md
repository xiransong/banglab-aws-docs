# Documentation Todo: Owner-Based AWS Resource Rules

This todo list tracks the documentation updates needed for the May 10, 2026
AWS permission model change.

The new model assumes that multiple researchers may share the same AWS member
account. Each researcher has a canonical username based on their full name, for
example `xiransong`, `alicezhang`, or `bobchen`.

Under the new `EC2-GPU-Operator` permission set:

- EC2 instances are isolated by the `Owner=<username>` tag.
- EBS volumes are isolated by the `Owner=<username>` tag.
- Security groups should be created with the `Owner=<username>` tag.
- S3 buckets are isolated by bucket name: `banglab-<username>-*`.
